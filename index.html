Projects: Programming

Notescribe (WIP)

Started at Hack The North 2019, Notescribe, is a web service that helps students easily transcribe and annotate audio from lectures.

Aided by Google's Speech-to-text API, students and staff only need to make minor changes to make readable notes from a recording. Notescribe is built in Flask (Python) and Google Firebase for the back-end, and utilizes Bootstrap for the front-end.

Programming (p.2)

Hawkshot

This was our group of 3's submission for the Riot Games Legends of Runeterra API Challenge.

Hawkshot is a social web service that allows players of the game Legends of Runeterra to create and vote on hints on how to play the game and improve.

Users can create an account through Google Firebase to access the website's voting and creation features. I built the back-end API in Flask using Google Firebase for authentication and storing user and hint data. The front-end was built in React, and the in-game component was built in Electron.

You can try it here: http://hawkshot.herokuapp.com/

Hackathons + Contests

F Speaking: the F stands for Fix

My group's submission for Hack the 6ix 2018 was a web application that could help people practice their public speaking. Using Google's Speech to Text API, we compared someone's speech with their original text and suggest things they could improve on. We built the application with Node.js, using p5.js and CSS to draw objects on screen.

Insurance Informatics

At Hack The North 2018, my group made a program that could read insurance data and dynamically display demographic data using dc.js. These graphs can be interacted with and filtered to a user's selection.
